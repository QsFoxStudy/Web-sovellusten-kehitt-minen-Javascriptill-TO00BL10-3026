## Tietoja sovelluksesta
[Musiikkitietosovellus] on albumi-info sovellus joka hakee 6 ennalta määrätyn artistin albumikuvan, -nimen, -Kokonaiskuuntelukerrat ja tarjoaa linkin last fm artistin albumisivulle.
Tiedot ovat reaaliaikaisia LastFm rajapinnasta.

## Teknologiat

Käytin seuraavia teknologioita: `html`, `css`, `javascript`. Html luo graafisen ulkonäön yhdessä css:n kanssa ja javascript luo dynaamisia toimintoja JSON:in avulla hakien reaaliaikaiset tiedot Last.fm api:n kautta.
